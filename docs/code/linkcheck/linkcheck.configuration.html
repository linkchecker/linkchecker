linkcheck.configuration.get_user_config()[source]¶

Get the user configuration filename. If the user configuration file does not exist, copy it from the initial configuration file, but only if this is not a portable installation. Returns path to user config file (which might not exist due to copy failures or on portable systems). :return configuration filename :rtype string
